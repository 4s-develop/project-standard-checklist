# 運用・監視 チェックリスト（詳細版）

## 1) 監視ダッシュボード/SLI

- 目的: 稼働状況の可視化
- 手順: SLI 選定 → ダッシュボード構築 → 共有
- 成成果物: ダッシュボード

## 2) アラート設計

- 目的: 障害検知の自動化
- 手順: 閾値/通知/担当/エスカレーション定義
- 成果物: アラート設計書

## 3) インシデント対応手順

- 目的: 迅速な復旧
- 手順: トリアージ → 連絡 → 回復 → 事後レビュー
- 成果物: インシデント記録

## 4) WAF 運用

- 目的: ルールメンテ/誤検知対策
- 手順: ログレビュー → ルール調整 → 再検証
- 成果物: 運用記録

## 5) バックアップ運用

- 目的: データ保全
- 手順: 頻度/保管/復元テスト
- 成果物: 運用記録

## 6) 運用レビュー議事録

- 目的: 継続的改善
- 手順: 月次レビュー → 議事録 → 改善計画
- 成果物: 議事録
