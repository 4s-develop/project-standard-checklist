# 総合テスト チェックリスト（詳細版）

## 1) 総合テスト仕様/項目/結果

- 目的: システム全体の動作確認
- 手順: シナリオ/動線/例外/回帰項目を定義 → 実施 → 結果記録
- ツール: TestRail/Excel/Playwright
- 成果物: 試験仕様/結果
- 受入基準: 重大不具合なし、残存は許容範囲
- 証跡: レポート

## 2) 性能/負荷試験（SLI/SLO）

- 目的: 目標性能の満たし確認
- 手順: 指標（p95, error rate）定義 → 負荷施行 → 結果評価
- ツール: k6/JMeter
- 成果物: 性能試験レポート
- 受入基準: SLO 達成（または緩和策）

## 3) 脆弱性診断 + SCA/OSS ライセンス

- 目的: セキュリティリスクの検出/OSS 遵守
- 手順: 外部診断/自動スキャン → 結果レビュー → 対応
- 成果物: 診断報告/対応計画
- 受入基準: High/Critical 解消

## 4) アクセシビリティ（AA、範囲調整可）

- 目的: 主要画面の利用可能性確認
- 手順: キーボード/コントラスト/代替テキストをチェック
- 成果物: チェックリスト
