# コンプライアンス（必要時のみ） チェックリスト（詳細版）

## 1) APPI 適合（PII がある場合）

- 目的: 個人情報保護法遵守
- 手順: 収集/利用/保存/破棄ポリシー記載
- 成果物: 適合チェック表

## 2) OSS ライセンス遵守

- 目的: 著作権/ライセンス遵守
- 手順: 依存一覧/第三者ライセンス通知方法を定義
- 成果物: ライセンス一覧

## 3) アクセシビリティ適合報告（公開 Web 時）

- 目的: JIS/WCAG 準拠報告
- 手順: 試験 → 結果まとめ
- 成果物: 適合報告書

※ Cookie/同意 UI は現時点では不要（案件要件に応じて将来追加）
